*HTML*

(Hyper Text Markup Language)

        * HTML is standard markup language for creating web pages.
        

        * HTML stands for Hyper text Markup language.
        

        * HTML describes the structur of web page.
        

        * HTML consits of a series of elements.
        

        * HTML elements tell browser how to display the content.
        

        * Tim-Berners-Lee specified HTML and wrote the browser and server software in late 1990.
